# Getting Started

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the local dev server:

```bash
npm run dev
```

The site will be available at `http://localhost:4321`.

## Build

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

This project deploys automatically to GitHub Pages via the GitHub Actions workflow (`.github/workflows/astro.yml`) on every push to `master`.
